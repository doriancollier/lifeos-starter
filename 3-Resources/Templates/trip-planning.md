## Pre-Travel Checklist

### 2+ Weeks Before
- [ ] Book flights
- [ ] Book accommodations
- [ ] Book rental car (if needed)
- [ ] Request time off work
- [ ] Check passport expiration (6+ months validity)
- [ ] Research visa requirements
- [ ] Start packing list

### 1 Week Before
- [ ] Confirm all bookings
- [ ] Arrange pet care / plant watering
- [ ] Hold mail or arrange pickup
- [ ] Download offline maps
- [ ] Check weather forecast
- [ ] Notify bank of travel (avoid card blocks)
- [ ] Print/download confirmations
- [ ] Charge devices and pack chargers

### Day Before
- [ ] Pack luggage
- [ ] Check in for flights
- [ ] Confirm transportation to airport
- [ ] Set out-of-office (if needed)
- [ ] Take out trash
- [ ] Check all doors/windows locked

### Day Of
- [ ] Grab snacks/entertainment
- [ ] Verify ID/passport
- [ ] Check flight status
- [ ] Leave on time!
